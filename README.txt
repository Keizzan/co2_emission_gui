Project Name: 
Emission App 

Synopsis:
GUI program, which read co2 csv file, validates it and allow user choose fossil fuel and country to plot charts.

Application requirement: 
    Python 3.8.10,
    pip 20.0.2,
    tkinker 8.6
    external python3 libraries are (you can install them, by running command 'pip install -r requirements.txt'): 
    cycler==0.11.0
    fonttools==4.33.0
    kiwisolver==1.4.2
    matplotlib==3.5.1
    numpy==1.22.3
    packaging==21.3
    pandas==1.4.2
    pandas-schema==0.3.6
    Pillow==9.1.0
    pyparsing==3.0.8
    python-dateutil==2.8.2
    pytz==2022.1
    six==1.16.0


For End-Users
-----------------
    1. Run app.py file,
    2. Choose country of interest
    3. Choose carbon fuel / pollutant
    4. Plot chart
